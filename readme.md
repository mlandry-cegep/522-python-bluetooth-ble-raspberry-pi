# Packages installés 

> J'espère ne pas en avoir oublié! ;)

```bash
sudo apt-get install libbluetooth-dev
sudo apt-get install -y libusb-dev libreadline-dev libglib2.0-dev libudev-dev libdbus-1-dev libical-dev libdbus-glib-1-dev
sudo apt-get install python3-pip python3-dbus
sudo pip install --break-system-packages git+https://github.com/ukBaz/python-bluezero.git
```
